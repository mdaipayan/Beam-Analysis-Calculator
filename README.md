# 🔧 Beam Analysis Calculator

A comprehensive Streamlit application for calculating Shear Force Diagrams (SFD) and Bending Moment Diagrams (BMD) for various beam configurations.

![Beam Analysis Demo](demo.png)

## Features

- **Multiple Beam Types**: Simply Supported, Cantilever, Fixed-Fixed, Overhanging
- **Load Types**: Point loads, UDL, Concentrated moments
- **Real-time Visualization**: Interactive matplotlib plots
- **Export**: Download results as CSV
- **Key Metrics**: Automatically calculates maximum shear and moment values
